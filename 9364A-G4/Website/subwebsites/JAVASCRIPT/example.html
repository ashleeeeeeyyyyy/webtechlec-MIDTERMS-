    1. pop()
      • Remove the last element in the array
      Javascript File:

                                                        console.log("Remove number: "+pop.pop());
      Output: Remove number: 4
    2. push()
      • Add a new element at the end of an array
      Javascript File: 
                                                              var push = [1,2,3];
                                                        push.push(4);
                                                        console.log(push);
      Output: [1,2,3,4];
    3. shift()
      • Removes the first element in the array and moves all other elemnts to the lower index
      Javascript File:

                                                        console.log("Remove number: "+shift.shift());
      Output: Remove number: 1
    4. concat()
      • Creates a new array by merging existing arrays
      Javascript File:

                                                        console.log(array3)
      Output: ["1", "2", "3", 4]
