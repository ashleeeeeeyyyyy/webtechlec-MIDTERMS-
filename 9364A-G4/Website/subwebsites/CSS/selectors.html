  • Selectors
  • Selector Syntax
    1. Sequence of simple selectors separated by combinators.
  • Simple Selectors
  1. This selector selects any type of an element in an HTML page. It can only match a single element per selector. 
 *{
                                            color: yellow;
                                            background: black; /* every element must have a black background */
                                            }
  1. The ID selector is used to style a specific unique element. This selector uses the attribute of the HTML element. It is also defined with the hash character ("#") 
 
                                        #initial{
                                            background-color: grey; /* every element with the id "initial" must have a grey background*/
                                            border: green 2px; /*all elements with the selector must have a green border with 2px padding*/
                                        }
  1. This selector is used to style a group of elements. This is defined with a period (".")
                                        .middle{
                                            text-align: center; /*All elements or group of element with the class selector "middle" must have its text aligned to the center. */
                                            background-color: pink; /*All elements must have a pink background. */
                                        }
  1. This selector is used to select elements with a specific attribute.
                                            a[target] { /*This is a regular attribute selector. */
                                                color: skyblue;
                                            }

                                            a[target=_blank] { /* This is an attribute selector with a value. */
                                                color: skyblue
                                            }

                                            [title~="flower"] {  /*This type of attribute selector selects elements with an attribute which contains a certain word*/
                                                border: 4px solid green;
                                            }
  1. This selector matches elements by the node name it is given. 
 

                                        p{ /* All paragraph <p> elements must follow the following styles */
                                            color:green;
                                        }
  1. A combinator explains a relationship between selectors.
  • provides additional confition
  • used to impose additional matching constrains following:
  • [space] descendant combinators
    •  
                                                    div p { /*This is matches all elements which are descendants of a specific element*/
                                                    background-color: green;
                                                    }
  • Child combinators (>)
    •                                                     div > p { /* This selector selects all elements which are the immediate children of a specific element */
                                                    background-color: green;
                                                    }
  • Sibling combinator
    1. Adjacent Sibling Combinator (+)
      2.                                                         div + p { /*This sibling selector selects all elements that are adjacent siblings of a specific element. */
                                                        background-color:green;
                                                        }
    2. General Sibling combinator (~)
      3.                                                         div ~ p { /*This selects all elements that are siblings of  specific element.*/
                                                        background-color: green;
                                                        }
