HTML Stylesheets

  • 3 sources of styles

    1. Author styles
      • External (aka linked) Stylesheets
          • This is a CSS method which allows a separate style sheet(external) to be attached to the main document it is styling.
          A great advantage in using this kind of CSS is that it can be used by all documents which require it.
      • Embedded (aka internal) Stylesheets
          • This is a stylesheet which is embedded directly to the HTML file it is currently styling. This requires the <style> </style> element for the stylesheet to activate.
          This can be used throught the current document it is in, but cannot be used by other documents.
      • Inline styles
          • This stylesheet method as its name says, only styles the content it is currently coded to. This cannot be used by other content with the same elements however.
          An advantage to this type of method, is specifying the style of the content whcih requires it.
    2. User Styles

      • There are some browsers that provides alternatives throught extensions or plugins
      • Example: Stylish

    3. User Agent Styles
